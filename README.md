# Chemoinformatics: using AiZynthFinder

## Цель

Настроить ретросинтетический поиск с помощью [AiZynthFinder](https://github.com/MolecularAI/aizynthfinder) для каждой молекулы из SDF-файла.

## Задачи

1. Разработать числовую скоринговую функцию синтетической сложности

* На основе результатов работы AiZynthFinder необходимо предложить числовой скоринг синтетической доступности/сложности молекул.

Требования:

        - Скоринг должен быть не бинарным, а численным (например, чем выше значение — тем сложнее синтез, или наоборот — оговорите явно направление шкалы).
        - В скоринг рекомендуется включить метрики, доступные в AiZynthFinder.
        - Молекулы, для которых AiZynthFinder не нашёл синтетического пути, также должны ранжироваться по этой метрике (не должно быть одинакового базового значения для молекул без синтетического путем)
        - Чётко обосновать принятую логику расчета скоринга.

2. Рассчитать скоринговую функцию для всего датасета и оценить скорость её вычисления

Необходимо:

    - рассчитать выбранную/разработанную скоринговую функцию для всех молекул,
    - измерить и указать в отчёте время вычисления скоринга на весь датасет целиком.


## Ожидаемый результат

1. SDF-Файл с исходной информацией и рассчитанным скорингом (имя должно быть понятным, например SynthScore)

2. Код и инструкция по воспроизведению

* Код должен:
    * принимать на вход файл с молекулами 
    * запускать AiZynthFinder и собирать результаты ретросинтеза, рассчитывать скоринговую функцию,
    * сохранять итоговый SDF (или другой формат) с рассчитанным скором.
* Также необходимо приложить краткую инструкцию (в README или в отчёте) с описанием:
    * требований к окружению (Python-версия, зависимости),
    * установки AiZynthFinder (как вы это сделали),
    * команд/шагов для полного воспроизведения результатов.

3. Отчёт о проделанной работе (.docx / .pdf).

    3.1.	Краткое описание подхода:
    * как вы интегрировали AiZynthFinder,
    * каков общий пайплайн от входного SDF до итогового скоринга,
    * какие допущения/ограничения вы ввели.

    3.2.	Описание скоринговой функции:
    * формальное определение / формула,
    * какие факторы учитываются (число шагов, наличие билдинг-блоков, статус ретросинтеза и др.),
    * как трактовать значение (что означает высокий/низкий скор).

    3.3.	Примеры и анализ:
    * несколько конкретных примеров молекул:
        * «простых» (с низкой синтетической сложностью),
        * «сложных»,
        * без найденного пути — с разбором, почему так произошло по вашему мнению.
    * интерпретация скоринговых значений на этих примерах.
    3.4.	Оценка производительности:
    * время работы на датасете.
